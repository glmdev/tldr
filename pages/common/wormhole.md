# wormhole

> Share files from one computer to another, safely.

- Send a text message:

`wormhole send`

- Send a file/directory:

`wormhole send path/to/file`

- Receive a file/directory/text message:

`wormhole receive access_code

- Receive SSH keys:

`wormhole ssh invite`

- Send SSH keys:

`wormhole ssh accept access_code`
